## Wipe Old Mac
Do this only after verifying all data has been migrated to the new Mac.

- [ ] Sign out of iCloud (System Settings → Apple ID → Sign Out)
- [ ] Sign out of iMessage (Messages → Settings → iMessage → Sign Out)
- [ ] Sign out of App Store (App Store → Store → Sign Out)
- [ ] Deauthorize Apple Music/iTunes (Music → Account → Authorizations → Deauthorize This Computer)
- [ ] Unpair Bluetooth devices (keyboard, mouse, AirPods — so they pair cleanly with new Mac)
- [ ] Remove Mac from Find My (System Settings → Apple ID → Find My)
- [ ] Erase All Content and Settings (System Settings → General → Transfer or Reset → Erase All Content and Settings)
  - This handles disk wipe, Apple ID sign-out confirmation, and macOS reinstall in one step (macOS Monterey+)
